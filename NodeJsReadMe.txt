1. To initialize and install a default package:
npm init

2. To install the internal file directory for initialization as "express" and save
npm install express --save

3. To maintain auto refresh of the package
npm install nodemon -g

4. To install the express-package(same as step 2) express-validator to validate the form
npm install express-validator -- save

5. To install the express-package(same as step 2) express-session package
npm install express-session -- save

6. To install the ejs package the form
npm install ejs -- save

7. To include DB package
npm install ibm_db --save



8. run below to give access to db.
export DYLD_LIBRARY_PATH=/node_modules/ibm_db/installer/clidriver/lib/icc

9. run below to start the service
nodemon
